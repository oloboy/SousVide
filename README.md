# SousVide (PWA)

Questo repository contiene la web app "Sous-Vide Time & Safety Calculator" configurata come PWA.

## Avvio locale
- Serve un piccolo server HTTP (i browser bloccano i moduli ES da `file://`): es. `python -m http.server 8080` oppure `npx http-server -p 8080 .`
- Poi apri `http://localhost:8080` nel browser.
